vz_wiki Models¶

Page¶

class models.Page¶

Pages are the main concept of a wiki.

Fields¶

class models.Page

Page objects have the following fields:

title¶

Required. Title is the main identifier for the page.

  • CharField
  • max_length is 255
  • must be unique
slug¶

Required. Slug is used for the URL for each page.

  • SlugField
  • must be unique
tags¶
Required. A TagField. See Django-Tagging.
creator¶
A User from Django’s auth
is_editable¶

A page can be locked to disable editing. This is an administrative function for those what if? situations.

  • BooleanField
is_checked_out¶

A page is locked once checked out.

  • BooleanField
created_on¶

Datetime object for when page was created.

  • DateTimeField

Methods¶

class models.Page

Page objects have the following methods:

check_out(user)¶

Checks out a page

Parameter:user – User object that is checking out the page
